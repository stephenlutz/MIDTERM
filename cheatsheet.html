<!DOCTYPE>
Defines the document type

<a>
Defines a hyperlink
href, hreflang, media, ping, rel, target, type

<abbr>
Defines an abbreviation

<acronym>
Defines embedded acronyms

<address>
Defines an address element

<applet>
Defines an embedded applet

<area>
Defines an area inside an image map
alt, coords, href, hreflang, media, ping, rel, shape, target, type

<audio>
Defines sound, such as music or other audio

<b>
Specifies bold text

<body>
Defines the document's body, containing all the contents

<br>
Inserts a single line break

<button>
Defines a clickable button

<center>
Used to center-align text

<div>
Defines a division or section

<font>
Specifies a font face, size, and color

<form>
Creates an HTML form for user input

<h1> to <h6>
Defined as HTML headings, from most to least important

<i>
Specifies italic text

<img>
Inserts an image on the page

<li>
Defines a list item

<link>
Links to an external resource/document

<ol>
Defines an ordered list

<p>
Defines a paragraph

<q>
Defines a short quotation

<script>
Defines a client-side script (Javascript)

<section>
Defines sections in a document

<source>
Specifies multiple media resources

<strike>
Defines strikethrough text

<style>
Defines style information

<table>
Defines an HTML table
<td>, <th>, <tr>

<title>
Required in all HTML documents and defines the title
